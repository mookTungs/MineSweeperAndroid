# MineSweeperAndroid
- goolgle play: https://play.google.com/store/apps/details?id=com.mookt.minesweeper
- classic minesweeper game
- 3 level of difficulties
  - easy 10 mines
  - intermediate 40 mines
  - advanced 99 mines
- long pressed for flag
- ability to zoom in and zoom out
- has stopwatch
